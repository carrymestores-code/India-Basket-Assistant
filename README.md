# India Basket Finder

India Basket Finder is a static prototype website for finding trusted India grocery routes by city, budget, category, and grocery need.

## Files

- `index.html` - website homepage
- `styles.css` - saffron India theme and layout
- `app.js` - route logic, opportunity records, and local assistant
- `.nojekyll` - helps GitHub Pages serve the static site directly

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `.nojekyll`
   - `README.md`
3. Open the repository settings.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Save.

GitHub will provide a public website link after deployment.

## Master Rules

- No personal emails, accounts, UPI IDs, or private data inside the app.
- Zero-cost first.
- Use public routes and public source links only.
